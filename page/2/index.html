一个梦

发表于 2007-03-13 分类于 blogspot

今天早上作了一个梦，关于思思的，其实昨天也梦到了，不过当时没有纪录，就忘了很多，不完整了，隐约记得思思是那么美丽，那么惹人怜爱，笑容还是那样无暇。今天这个梦却让我很紧张，都说梦和现实是相反的，希望如此就好了。梦大致是这样的：

在一间很大很豪华的屋子里，我，思思，还有一个不认识的人，连样子都忘记了或是这个人本就没有样子吧，不知道那里来的。房子中间有一个很大的类似磨盘的“床”。开始我很想和思思在一起说说话，可是她一直都不怎么理我，我很无奈，就跑到床上去躺着。一会儿我听见思和那个人在说话，说的什么我不知道，我探着脑袋看了一下，她们是抱在一起的，思已经变成了一个很抚媚的女人，严重充满了柔情。我把脑袋缩回来，受不了她们，才一会儿，我听到一声巨响，我看过去，看到那个男的要去开房门，大概要走了，思她拿起一副杠铃，自己向后倒去，结果杠铃压在了她胸口，她是想以此威胁那个人，不让他离开。在看去思思她已经开始吐血，我脑袋轰一声，顿时觉得天旋地转，我赶紧跳下“床”，抱起思思开始往外跑。思她穿的一声洁白的连衣裙，胸口已经染上了血。

我心里很荒，开始往楼下冲，要找医院，我力气不够，抱着她十分的吃力，可是下了一层又一层，不知道要下多少层楼，手感觉快断掉了，可是我就一直这么抱着她跑。思她眼里充满了幽怨，我已经快哭了。最后终于下到楼底，我想打的，可是发现现在是深更半夜，根本没有车，突然前面开来了两辆电动三轮车，外形改装成QQ车的那种。我选了一辆，要上车时我发现那个男的也跟来了，他抢先一步坐了车后排的座位，我只好也将思放在后排的座位，然后自己和司机去挤，座位都很小的，而且才发现那种车没有车门，如果车子太晃很可能把人给扔出去，我跟司机说“你开慢点，后面那位小姐身体不舒服，车子不能太晃了”，那司机是个很年轻的小伙子，很乐于说话那种，说好的。然后车子开始开动。可是路面怎么全都是泥巴路，坑坑洼洼的，车子晃的很厉害，我生怕万一思被甩出车外了，更何况此时发现路的两边都是悬崖，而且路很窄，基本只能过这一辆车，有时候那路甚至左右倾斜的很厉害，车子整个都是斜的，不过还好我们都没有被甩下车。车子开的很可怕，随时都可能翻车，我一直催司机不能开的太快，别太晃，可是路面却是很烂，最后我说我来开吧，然后我自己亲自来开，可是很难开，有三次遇到路面比车轮距还要窄，可是车速很快只能强行冲过去，结果都安然无恙。

最后车子开到了一个地方，一个小土坡的样子，很象我家乡的一处土斜坡，我们都下车了，前面车子开不进去了，我要把思她抱到医院去，我问司机多少钱，想来恐怕要二三十，最低也要十块，可是他说两块钱，我一掏钱结果发现只有一张五十的，我正考虑，结果发现周围多出几个人，好像都是我的朋友但是我一个都不认识，后来争着付钱怎么的耽搁了一阵，我抱起思思开始往医院走，到一片竹林的时候一个护士打扮的中年妇女告诉我们医院的地方，我们继续前进，现在抱着思思感觉越来越轻松了，不象先前那么累了，思她好像已经好很多了，身上的血迹也不见了，还微笑了，可是始终就没有看我一眼，仍然很牵挂我身后那个人，我心痛的要命，不管别的，先把她送到医院。思她感觉好像已经没有什么事了，可是我想受了那么重的伤是一定要去医院检查治疗的。

一直是我在前面带路的，其实我并不清楚医院的具体位置，好像以前去过，凭记忆走，结果后来身边一个小丫头提醒我应该走另外一条路，我记得好像是走这条路的，可是我又想起那次和思思逛步行街回来的时候我要带着她走路回去，结果走了半天发现一直绕一个环形的地方兜了一个圈子，从此思思把这当成笑话来取笑我，我想这次也许我又错了，后来也不知道怎么走啊走，终于到医院了，医生开始诊断，问我们情况，我说她伤的很严重都吐血了，后来就记得不是很清楚了。

总之整个过程思思她都没有看过我一眼，好像无视我的存在，而她一直还是牵挂着那个让她受伤的那个人，我想也许日有所思，夜有所梦吧，但是我真的很怕以后会有这么一个人让思她受伤害，如果思思她以后的男朋友对她不好，我会杀了他。思思，你一定要过的好好的，千万不要受伤害。。。。

以后，我都想把关于你的梦纪录下来，虽然有时候乱七八糟很不靠普，但是忘记了是一种可惜。

失败的16项主因

发表于 2006-11-17 分类于 blogspot

生活中最大的悲剧就是人们热切地尝试却履遭失败，之所以说是悲剧，就在于和少数成功人士相比，失败的人数占压倒性的大多数。

1） 没有明确的生活目标。没有奋斗的中心目标或明确的努力主向，就没有成功的希望。

2） 没有非同寻常的雄心抱负。如果对凡事漠不关心，不想在人生中求发展，不愿付出代价，那么这样的人也将成功无望。

3） 缺乏自律。纪律来自自我控制，这意味着人必须控制所有的消极思想，只能先控制自己，才能控制环境。自制是人类面对的最艰巨任务，如果无法战胜自我，就会被自我征服，

4） 拖拉。这是失败最普遍的原因之一，拖拉“老人”存在于每个人心中的阴暗角落，正是因为一直都在等待“适当时机”，才开始做那些值得做的事情，其实时机永远不会“适当”。

5） 缺乏毅力。不管做什么，大部份人开始时都满怀信心，但却不能善始善终。大部份人一遇到失败就容易放弃。毅力是不可取代的。把毅力当座右铭奉行到底的人，发现失败会自行退下，失败是无法对抗毅力的。

6） 消极的个性。因为消极的个性，而将别人拒于千里之外者，不会有成功的希望。成功来自力量的运用，而力量又来自与他人的合作，消极的个性无法促成合作。

7） 无法克制“不劳而获”的欲望。这种机体本能使很多人走向失败。

8） 缺乏果断的决策力。成功的人士会果断决策，然后如果有必要，再慢慢改进。失败者往往花很长时间才能作出决策，但很快就需要修改，而且要频繁修改。犹豫和拖拉是一对双包胎，只要找到其中一个就一定能找到另一个，所以必须趁它们没有将你完全束缚在失败的车轮上，果断地把它们消灭。

9） 过度谨慎。不主动抓住机会的人往往只能捡别人挑剩的机会，过度谨慎和不够谨慎都不可取，人生本来就充满偶然成分。

10） 迷信与偏见。迷信是恐惧的一种形式，也是无知的表现，成功人士心胸宽广，无所畏惧。

11） 目标不专。“万事通，万事松”，要全心全意专注于一个主要目标。

12） 缺乏热情。没有热情，一个人就没有说服力，而且热情有一种感染力，一个人如果拥有热情，并能适当控制热情，往往会受到人们的欢迎。

13） 偏执。心胸狭窄很难取得任何进步，偏执说明一个人不积极获取知识。

14） 蓄意不忠。诚实是一种不可替代的品质，受无法控制的环境所迫，一个人可能一时不忠诚，也不会带来永久的破坏。但是，如果一个人蓄意不忠，则无药可救。他的得行为迟早会被发现，他负出的代价可能是失去信誉，甚至失去自由。

15） 自私和虚荣。这些品质问题好比亮起红灯，让人不敢靠近，是妨碍成功的致命因素。

16） 猜测而不思考。多数人往往漫不经心或者过于懒惰，不愿费心获取用于准确思考的事实。他们喜欢根据猜测或仓促得出的“结论”了事。

大学，我毕业了

发表于 2006-11-17 分类于 blogspot

这段时间我每天都在思考同一个问题：我还在学校呆着干嘛？

我很庆幸自己能来到大学的殿堂，在这三年里我学到了好多，学会了怎样去生活，怎样去做人。我有我的梦想，在大学，我终于知道该怎样去实现它！学校里大家都积极向上，每天都是生机勃勃的气象，大家充满干劲，几乎每个人都在努力实现自己的梦想，构建自己的未来。

大学里真正学习知识的地方是在除了教室的任何地方，可惜也有好多大学生并不知道。教室里学习的都是专业知识，那些都是用来做事而不是做人的。要做事比做人简单很多，所以大家不应该搞不清楚重点。做人的学问太大，我不是哲学家我也说不清楚怎样才算做人，但是我很清楚除了要实现自己的梦想之外更要学会如何去奉献。也许有的人会笑我活的太天真，说我不懂社会的现实。很多人觉得“人不为己，天诛地灭”，我很想问一句当你衣食无忧，家庭美满，事业有成的时候你还需要什么？我知道，你也许还没有想过那么远呢，因为你觉得自己永远达不到使自己满足的时候。如果你相信“衣食无忧，家庭美满，事业有成”对自己来说是件简单的不能再简单的事情，那么应该会想到剩下的就是回馈社会了。

现在我们的生活确实不用担心吃饭的事情了，但是地球上确实还生活着10几亿衣食不保的贫民啊，他们只能说老天的不公平，特别是那么多失学的小孩子，他们从一开始就输在了起跑线上，他们真的需要帮助啊。

想帮助别人首先要自己有能力，能力越大能帮助的人越多，我想这就是我人生的动力。这动力永远没有枯竭的时候，地球这么大，想人人都过上幸福生活谈何容易，但是只希望受苦的人尽量的少一点。

现在虽然忙起来了，但是生活变的有节奏了，不再颓废了。

日子总是要过的，开心也是一天，不开心也是一天，那自然要选择开心了。就算不开心，那也有很多可以让自己变的开心的办法。我和我的同学们都还呆在象牙塔里，我很不想出去，但是又很想出去。有很多理想等着我去实现，很多责任等着我去承担。人总是要长大的，似乎我已经长大了，可是还有很多不成熟，如果一年的时间可以好好利用起来的话应该是足够的，足够长足羽翼。

我很庆幸没有学习的压力，没有家长的指责，没有世俗的羁绊，最重要的是没有感情的纠葛。这些，我不光要感谢我的家人对我的理解和支持，朋友对我的关心，更要表扬我自己。现在，我已经很确定我将来要走的路，虽然即将面对很多尴尬，但是都已经无所谓。短暂的失败也是必经的过程，我已经做好准备。

我的同学都是很可爱的，和他们相处的时日已经不多，如果将来分开，我衷心祝愿他们万事如意，永远过得开心。还有跟我要好的朋友，更是如此。

走自己的路，让别人打的去吧！

我的爱,为什么很苦涩?

每个人都告诉我,爱,是幸福的,可是我似乎从来也没感觉到,也或许每次都只是那么一刹那的开心.我从来没想过爱一个人会这么让人心碎,是不是我原本就不该爱?我承认爱情上我很自私,我不能伟大到爱一个人不去计较任何付出.可是我爱得如此深我已不能自拔,明知道没有结果的爱情我还是要一头扎进去,犹如飞蛾扑火一般,最后落得遍体鳞伤,为的只是面对我的感情.我很不情愿承认自己在爱情面前是如此的懦弱,可是我又是如此的执着.我很害怕,害怕受到伤害,可是我从来不会避开.我不知道这样子到底算不算蠢,唯一支撑我的那是我不切实际的一点奢望.

每当我睡醒后想起刚才梦中似乎牵过你的手,今天的一切都会变得如此美好.我比较喜欢自欺欺人,这样可以抚慰我一时的伤痛.我知道,你爱的不是我,所以我不能把自己放到不该放的位置.我喜欢跟你聊天,哪怕看到手机上你给我的几个字我都很高兴了.回忆跟你在一起的日子其实已经成为一种习惯,尽管是那么的苦涩.我也曾经一度想忘却,但是我很失败…我也期待某天能有另外一个女孩出现在我的生活,可是更害怕会遇到第二个你…

在别人的眼中我永远都是一个天天开心的小子,我尽量的掩饰和抛开那些不如意的事情.我最害怕不开心的时候,所以总是极力的寻找值得开心的事情,我永远不要看悲剧电影,最最最讨厌!我宁愿所有人把我当小孩子看,甚至有人发现我躲在角落里流泪的时候也不会觉得奇怪.作为一个男生我很失败,我只能在没有人的地方数着内心的伤痕,那都是自己一道一道划上去的.我从来没有怪过任何人,只是觉得自己很没用.我能做什么呢?爱情就是这么残忍,此时我已经经不起摧残,我只希望能这么拖下去都行.曾经有朋友告诉我,过去的爱情不要再奢望可以找回来,没有意义了.是,她说的对,可是我狠不下心.

其实这么多的痛都是自己给的,我不怪谁,我也不怪自己.只是不懂,为何爱的这么痛.
