考试完后的想法

发表于 2006-05-27 分类于 blogspot

啊～今天考了大学法律和体育笔试，如果老师能保佑我的话应该可以过的吧，饿的神呀(此句明言出自《武林外传》－佟香玉)！ 现在学习都成了我的负担了，好讨厌这样！本来大学应该是很轻松的，不过现在被我糟蹋的…

唉～没办法，有得必有失嘛，我选择了“黑道”那么必须要有所付出的^_^。

这里解释一下什么叫“黑道”：其实是从我们辅导员郑昭莉老师那里听来的，话说大学想有一番作为的话有三条路可以走：1.白道；2.红道；3.黑道

1.白道，走科研路线。走白道之人适合那些不擅长交际，不喜欢世俗生活，或者暂时不希望走入社会的人。他们只要好好把书念好，学号课本知识，成绩考的顶呱呱的就行了，不用考虑那么多，目标都不用定，一个劲的读，一个劲的往上考就行了。不过也挺艰苦的，学士上面有硕士，硕士上面有博士，博士上面还有博士后，同学开玩笑说博士后上面就是圣斗士了，圣斗士还分青铜圣斗士，上面还有白银圣斗士和黄金圣斗士，不够的话还有雅典娜也加上去-_-“…

2.红道，就是从政。那么就把文科方面搞扎实一点，先考个公务员，然后慢慢向上爬，终极目标就是坐上总书记位置，那叫一个NB。

3.黑道，这个是重点，就是从事商业活动，当个大老板，开个大公司^_^，这个就是我选择的路线了。选择了这条道路的人付出的最多，谈起某某公司的老板，哪个都会很看得起，都知道要做老板不容易啊。具体我到还谈不出个所以然，至于我能谈出来的时候恐怕也是在我成功之后了吧^_^。 要做的事情太多了，因为这学期才刚刚建立的意识，我准备再花一年的时间来做准备。一年之后一定会有所改变，大好基础，做好迎接挑战的准备！目前写不出什么像样的东西，准备利用暑假的时间完完整整的梳理一下自己，把自己研究透彻，一一改正到符合“商业标准”^_^

其实我老是谈要成功要怎样，那都只是表面上的，我就是想证明自己不是那么平凡，“天生我才必有用”我相信这个！不在乎这句话的人甚至不相信这句话的人通常都只想过平淡的生活，我并不认为就不好，人嘛，各有所爱，有时候我也想安安份份的过日子，但是我又是农民出生，没办法过那种奢侈的生活，趁着还有几十年好活，那就痛痛快快的赌一把，就算输了又怎样了？大不了再来！跟人生赌博根本就不要赌本，我怕谁呀！哈哈… 最后再说说今天的琐事吧。

下午去给学工部一位老师修电脑，说是不能上网，老师本人不在，是一位同学请我去的，注意这里是请我去的哦。我在那里弄了一个小时吧，还重装了系统，最后怀疑是网卡有问题。本来今天考试完了没什么重要的事情我还打算弄不好就不回去的，结果发生了以下的事情：一位可能在学工部享有什么“高级”职称的学生刚进来不认识我，看到我在捣鼓老师的电脑，说：你干什么呀你，弄那个干什么？搞得我都回不过神来了，请我来的同学赶忙解释说我是网络中心的，那刚来的学生“哦”了一声，跑来看我怎么弄，一看桌面变干净了，马上对着我说：你重装系统啦？是不是！我靠，我算是明白这个学生的官架子了，我说我来的时候已经刚刚重装过了，我是以为装的系统盗版的不好造成软件或硬件冲突，所以又用我的盘重装了一次(这是事实！)他好像以为我在撒谎，问旁边的同学：他说的是不是真的，装系统是谁批准的？旁边那个同学开始是他装了一遍。然后这个“大官”又对我说C:盘那么多东西你给我弄哪里去了，啊？我真是强忍住才没有说话，我怕我说出一个字来就会发作！旁边的同学赶紧解释说东西都转移到其他盘了。他还是不相信，抢过鼠标在各个盘符里找，嘴里还不停了念：在哪里！找了找之后他开始研究不能上网的问题，我很有礼貌的说：你坐下操作把，然后我把位子让给他。他坐下后就在哪里捣鼓，看来好像还懂得电脑的一些基本操作，弄了一会后他说：以前也出现过这种问题，过两天就会好了。我听了真想马上就走，什么意思？我还是忍了，告诉他可能网卡有问题，因为一启用网卡就会造成卡机现象，很像硬件冲突的症状，但是并非是硬件冲突。他又重复的捣鼓了一遍，最后得出结论：不是网卡的问题，过两天就会好的。旁边的同学都很为难了，我想算了，遇上这种人算我长见识了，那么我说：好的，那就先等两天吧，如果没有好再找我吧。然后我就走掉了，还听见后面请我来的那个同学不停的叫我等一下，唉，我生气嘛，所以头也不回就走了，后来想想好像不应该对他发脾气啊。

后续：我回到网络中心和朋友发短信聊天，也说到这件事，我跟朋友说他最后还是得找我才行，并不是我夸张，学校能解决这种问题的人实在太少了，网络问题通常都只会向网络中心求助，可是网络中心的事情基本都是我在负责啊。后来才过了一个小时就接到有礼貌的那个同学的电话，让我还是去帮忙吧，在电话里也跟我道歉了，告诉我那个很吊的人其实根本没什么技术，只是在学生会工作，他又认识所以不好说他。呵呵，后来我还是去帮他搞，又弄了2个多小时终于找到原因，原来被WindowsXP欺骗了，它自动识别安装的网卡驱动程序根本不正确，我强制安装了正确的驱动程序后终于搞定了。其实问题很简单，但是要找出问题就难了-_-“ 还有，今天朋友之间闹矛盾了，唉～我也积极的想帮助他们但是不知道其了多少作用呢，呵呵。

PS:本来这个帖子应该在27日完成的，现在都过了27日了，是28日0点过后了，呵呵写了5个多小时呢，当然是因为给学工部哪里弄电脑和调解朋友的矛盾呀呵呵。
还有这两天在用办公室的电脑挂QQ直播，呵呵(偷笑)。这两天QQ直播上面看到“两只鱼组合”，感觉很可爱的两个来自集美大学的女生，虽然很多人骂她们疯癫，我倒是蛮喜欢的^_^。

成功的开篇－新开Blog的预言

发表于 2006-05-26 分类于 blogspot

其实很久就想弄个自己的博客，本来打算自己写一套博客程序的，但是真的是由于时间不够，要写出来估计黄花菜都凉了。要找个有归属感的blog空间也还费了我几个小时呢，一直以来我都比较崇拜Google，发现Google在提供这样的服务我当然作为首选啦。

我也正在积极地向Google两兄弟学习，希望哪天能像谢尔盖•布林(Sergey Brin)和拉里•佩奇(Larry page)那样获得成功。我是个不甘寂寞的人，希望能将自己的能力尽可能发挥出来，首先是为了证明自己(也许更应该说是满足虚荣)，将事业发展壮大；之后我就有资本有能力对全世界宣布我最终的理想了：为全人类创造财富！真的，我是这么想的，但是恐怕我现在说了也没人会相信，因为人们总是将一个普通小伙子的伟大抱负说成不切实际或者不自量力要不就是以为他在开玩笑。

我一直相信每一个人生来就具备实现这个理想的能力，只不过并不是每个人都会付出相应的代价去去换取。但是总有人能成功吧，凭什么我就不能呢，呵呵。很多成功的人甚至是白手起家，真正的白手啊，一无所有都可以啊。我呢，至少目前也读到大学了，算是具备了一点点知识，都不算白手了，我是这么认为的，那么我成功的机会相应会大一些的。

假如我给成功定个分数的话，那么我认为有希望成功的想法可以给5分，有目标可以给10分，敢于说出自己的理想可以给15分，把每天为此目的而奋斗的历程写下来可以给20分，能坚持不懈可以给30分，剩下20分留给智慧和机遇吧，那么100分的满分我目前都稳定收到了50分了，坚持不懈的分数我相信我可以得全30分，只是目前给出分数比较不客观，那么只要在智慧和机遇的20分里我挣到10分的话就及格了。

呵呵，想成功－就是这么简单^_^
